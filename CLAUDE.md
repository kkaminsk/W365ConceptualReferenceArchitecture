# Project Context

This repository contains a Windows 365 Cloud PC Reference Architecture document authored by Kevin Kaminski (Microsoft MVP).

## Repository Structure

- **W365Design1.0-Signed.pdf** - The main deliverable; a 98-page reference architecture document
- **Source/** - Editable source materials
  - **W365Design1.0.docx** - Word document used to generate the PDF
  - **Graphics/** - Diagram and illustration PNG files
- **README.md** - Repository documentation
- **LICENSE.md** - CC BY 4.0 license

## Document Purpose

This is an educational reference architecture for deploying Windows 365 Cloud PCs in enterprise environments. It covers:

- Cloud-native identity with Microsoft Entra ID
- Automated provisioning via Windows 365 Provisioning Policies
- Unified management through Microsoft Intune
- Zero Trust security with Conditional Access
- Application delivery and servicing strategies

## Target Audience

IT professionals, cloud architects, and technical managers planning Windows 365 deployments.
